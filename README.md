# FuretaPikku

A tool that makes you instalock any agent you want, dodge the game, and check what side you are in Select Screen

you can contact me at my discord .imu_sama.

# how to use

1. First you have to download the compiled version and run it(**MAKE SURE THE GAME IS RUNNING WHILE USING IT**) or download and run main.py.
2. it will ask for the region (first lunch only) which will be one from the following (EU, NA, AP, BR, LATAM, KR).
3. then it will ask you for your preferred agent(Reyna, Brim, etc.....) or you can type dodge to dodge the game, also you can type ally to know what side you are on (Attacker, Defender) on Select Screen.
4. then you start a game and the tool will tell you when the agent is locked(usually during the countdown).
5. have fun :)
